## How to use Pystachio
#### 1. Download zip and locate folder.
#### 2. Open terminal or CMD and move into 'compiladores' folder
#### 3. Create .pyst file and move to 'compiladores' folder
#### 3. Run in Terminal: `python3 run.py {fileName}`
> Enter filename without extension
